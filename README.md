# Compress PDF Online

Working with large PDF files can be frustrating.

This repository contains resources, guides, and tools for compressing PDF documents online.

## Why Compress PDF Files?

Large PDF documents:
- are difficult to upload
- take more storage space
- are slower to share
- may exceed email size limits
